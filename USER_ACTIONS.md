# Можливі дії користувача в YourBar

Цей документ містить опис основних функцій та дій користувача в додатку YourBar.

## Діаграма дій

```mermaid
graph TD
    Start((Запуск додатку)) --> Onboarding{Навчання?}
    Onboarding -- Так --> OnboardingFlow[Пройти навчання]
    Onboarding -- Ні --> MainTabs[Головний екран: Вкладки]
    OnboardingFlow --> MainTabs

    MainTabs --> CocktailsTab[Вкладка Коктейлі]
    MainTabs --> ShakerTab[Вкладка Шейкер]
    MainTabs --> IngredientsTab[Вкладка Інгредієнти]

    subgraph CocktailsTabActions [Коктейлі]
        CocktailsTab --> CT_SubTabs[Перемикання підвкладок: Всі / Мої / Вибрані]
        CocktailsTab --> CT_Search[Пошук та фільтрація: Назва, Теги, Методи]
        CocktailsTab --> CT_Add[Додати коктейль]
        CocktailsTab --> CT_Details[Перегляд деталей коктейлю]
        CT_Add --> CT_EditScreen[Екран створення/редагування]
        CT_Details --> CT_DetailActions[Дії в деталях]
        CT_DetailActions --> CT_ToggleFav[Змінити рейтинг/вибране]
        CT_DetailActions --> CT_Edit[Редагувати]
        CT_DetailActions --> CT_Delete[Видалити]
        CT_Edit --> CT_EditScreen
    end

    subgraph ShakerTabActions [Шейкер]
        ShakerTab --> ST_Select[Вибір інгредієнтів]
        ShakerTab --> ST_Filter[Фільтрація: Всі / В наявності]
        ShakerTab --> ST_Shake[Кнопка Shake]
        ST_Shake --> ST_Results[Результати пошуку]
        ST_Results --> ST_ResDetails[Перегляд деталей коктейлю]
        ST_ResDetails --> CT_Details
    end

    subgraph IngredientsTabActions [Інгредієнти]
        IngredientsTab --> IT_SubTabs[Перемикання підвкладок: Всі / Мої / Покупки]
        IngredientsTab --> IT_Toggle[Перемикання наявності прямо у списку]
        IngredientsTab --> IT_Search[Пошук та фільтрація за тегами]
        IngredientsTab --> IT_Add[Додати інгредієнт]
        IngredientsTab --> IT_Details[Перегляд деталей інгредієнта]
        IT_Add --> IT_EditScreen[Екран створення/редагування]
        IT_Details --> IT_DetailActions[Дії в деталях]
        IT_DetailActions --> IT_Edit[Редагувати]
        IT_DetailActions --> IT_Delete[Видалити]
        IT_Edit --> IT_EditScreen
    end

    MainTabs --> SideMenu[Бічне меню: Налаштування]
    subgraph SideMenuActions [Налаштування]
        SideMenu --> SM_Toggles[Перемикачі: Гарнір, Замінники, Одиниці, Екран завжди ввімкнений]
        SideMenu --> SM_Options[Вибір: Початковий екран, Тема, Поріг рейтингу]
        SideMenu --> SM_Tags[Керування тегами]
        SideMenu --> SM_Data[Експорт / Імпорт даних та фото]
        SideMenu --> SM_Reset[Перезапуск навчання / Скидання даних]
    end
```

## Основні функції

### 1. Керування Коктейлями
*   **Перегляд списку:** Користувач може переглядати всі коктейлі, тільки доступні (Мої) або вибрані.
*   **Пошук та фільтрація:** Можливість швидкого пошуку за назвою та фільтрація за тегами (напр. "Класика", "Міцний") та методами приготування (напр. "Shake", "Stir").
*   **Створення та редагування:** Користувач може додавати власні рецепти, вказуючи назву, інгредієнти, кроки приготування, теги та додаючи фото.
*   **Вибране:** Можливість ставити рейтинг коктейлям. У меню налаштувань можна задати поріг рейтингу для відображення у вкладці "Вибрані".

### 2. Керування Інгредієнтами (Інвентар)
*   **Наявність:** Швидке перемикання статусу інгредієнта (є в наявності чи ні).
*   **Список покупок:** Можливість додавати інгредієнти до списку покупок для майбутнього поповнення бару.
*   **Брендування:** Можливість вказувати конкретні бренди інгредієнтів, які є в наявності.

### 3. Режим "Шейкер"
*   Дозволяє вибрати інгредієнти, які ви хочете використати зараз, та знайти коктейлі, які можна з них приготувати.
*   Результати показують як ті коктейлі, що можна приготувати повністю, так і ті, для яких не вистачає лише кількох інгредієнтів.

### 4. Налаштування та Дані
*   **Персоналізація:** Налаштування теми додатку (світла/темна/системна), одиниць виміру (ml/oz) та початкового екрану.
*   **Керування тегами:** Створення та редагування власних тегів для коктейлів та інгредієнтів.
*   **Резервне копіювання:** Експорт та імпорт даних бару у форматі JSON, а також резервне копіювання фотографій у форматі TAR.
*   **Навчання:** Можливість перезапустити тур додатком у будь-який момент.
